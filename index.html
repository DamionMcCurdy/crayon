Crayon.css

Because colors have a name

Crayon.css is a list of css variables linking color names to hexadecimal values. Provided for your favourite CSS preprocessor, being it Stylus, Sass or Less. 

@import crayon

body
  background piggypink
  color grape
  border 1px solid licorice

The above list counts 273 colors, with names sourced from Crayola.

I quickly got the full list from the CaptainRedmuff's repository UIColor-Crayola.

Images have been created by means of SVG filters, applied to random paths resulting from trigonometric functions. Have a look at this pen for further details.

SVG filters are adapted from this other pen by Dirk Weber.
